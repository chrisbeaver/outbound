# Contributing

## Reporting Issues

- Check existing issues first
- Include VSCode version, OS, and Laravel version
- Provide steps to reproduce

## Pull Requests

1. Fork the repo
2. Create a feature branch
3. Test in a real Laravel project
4. Submit PR with clear description

## Testing

- Test with Laravel 8+
- Verify route detection works with your changes
- Test in the VSCode extension host (F5)
